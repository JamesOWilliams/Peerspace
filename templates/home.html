What is Peerspace?

Add why would you care?

Been meaning to do that thing for ages?

Is there something you've been meaning to do for years but never got round to it? By posting commitments on Peerspace, everyone can see whether you have done it or not.

Want to collaborate with like minded people?

Need to find a gym partner? Or squash opponent? Make a post on Peerspace about it and like-minded people will get in touch, you might even find a future co-founder!

Want to get away from the clutter on Facebook?

Gone are the days of letting people know how you feel, Facebook has become nothing but a mass advertising platform with the occasional funny cat video. Peerspace is free of ads and unwanted clutter.

How does Peerspace Work?

A breif overview of the nuts, bolts and cogs of the site.

Powered By Django

Peerspace is built using the Django web-framework for Python. It allows developers to build powerful web applications quickly and efficiently. Django takes care of the boring stuff and enables the developer to focus on the important stuff, the application itself. This project was my first encounter with Django and I only have positive things to say, it has been very easy to pick up and the support community is broad and extremely helpful. Find out more here.

Postgres Won't Forget(Hopefully)

PostgreSQL is the memory of Peerspace, it remembers everything from user information to what I had for dinner last night, four years ago, to every post created in Peerspace history. Django's object relational mapper (ORM) makes it very easy to interface with the PostgreSQL database, this is one of my favourite features of Django. I have had experience quering a MySQL database with PHP, comparatively Django's ORM makes it a walk in the park. Find out more here.

Heroku is the Engine

Without Heroku my test user is forever trapped within the local development environment of my laptop. Heroku gives me the ability to share Peerspace with the world. I can choose any number of dedicated Heroku dynamos to deal with the number crunching and Python running as I need, this allows me to scale the app to meet any demand. Find out more here.

The codebase

The heart of the application. Download

The backend of Peerspace is powered by Django. Django is a web-framework for Python. I have always been interested in computer science, but since learning more about how the internet works I have become more and more interested in taking programs to the web. Django has given me the ability to build web applications that are scalable and fast.

Python powers the backend of Peerspace. I have enjoyed building this application in Python, the syntax encourages clean code and the Python containers are brilliant. The backend has an API that is consumable by the frontend. This allows easy scalling of the backend with a hosting service such as Heroku.

The content layer of the frontend is written in HTML5 and CSS3. I have used the Twitter Bootstrap 4 framework to build the site. Bootstrap allows for rapid frontend development and it saves a lot of time when building responsive webpages.

JavaScript powers the dynamic elements of the frontend. It is the communication layer between the backend REpresentational State Transfer (REST) API. Using Asyncronous JAvaScript and XML (AJAX) the frontend can GET and POST data to the database. This allows me to create responsive user interfaces and dynamic webpages.

Riot.js is a simple and elegent component-based user interface (UI) library. It powers the post feeds of Peerspace. It allows me to create custom HTML tags, for example every post that is loaded on Peerspace uses a custom HTML tag complied but Riot.js. The most useful feature in Riot.js (for this project in particular) is the ability to loop of data sets. In doing this I can add HTML to the document object model (DOM) dynamically and efficiently.
